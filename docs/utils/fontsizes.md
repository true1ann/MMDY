## Explanation

All fontsizes are exported as so:

- `.fs-1` - h1
- `.fs-2` - h2
- `.fs-3` - h3
- `.fs-4` - h4
- `.fs-5` - h5
- `.fs-6` - normal text
- `.fs-7` - small text
